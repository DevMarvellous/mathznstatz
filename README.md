# Mathz 'n' Statz

Welcome to the Mathz 'n' Statz project! This website is designed to provide users with a variety of mathematical tools and statistical resources, all while offering a visually appealing and user-friendly experience.

## Project Structure

The project is organized as follows:

```
mathznstatz
├── public
│   └── index.html         # Main HTML document for the website
├── src
│   ├── assets             # Directory for static assets (images, fonts, etc.)
│   ├── components         # React components for the application
│   │   ├── Navbar.js      # Navigation bar component
│   │   ├── ThemeSwitcher.js# Component for switching between light and dark themes
│   │   └── Animations.js   # Component for handling animations
│   ├── pages              # React components representing different pages
│   │   └── Home.js        # Homepage component
│   ├── styles             # CSS styles for the application
│   │   └── theme.css      # Styles for light and dark themes
│   └── App.js             # Main entry point for the React application
├── package.json           # npm configuration file
└── README.md              # Project documentation
```

## Features

- **Light and Dark Themes**: Users can switch between a light theme (blue and white) and a dark theme (black and blue) using a sun/moon icon.
- **Stunning Animations**: The homepage includes engaging animations to enhance user experience.
- **Responsive Design**: The website is designed to be responsive and accessible on various devices.

## Getting Started

To get started with the project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd mathznstatz
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```
